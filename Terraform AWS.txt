//////////////////////IAM Policies with terraform ////////

 Terraform + IAM policies is one of the most powerful combinations in AWS automation

Create an IAM policy
You define a policy document in Terraform using JSON syntax.

hcl
resource "aws_iam_policy" "developer_policy" {
  name        = "developer-policy"
  description = "Allow developers to manage EC2 and S3"
  policy      = jsonencode({
    Version = "2012-10-17"
    Statement = [
      {
        Action   = ["ec2:*", "s3:*"]
        Effect   = "Allow"
        Resource = "*"
      }
    ]
  })
}

Assign (attach) the policy
You can attach the policy to a user, group, or role.

Attach to a user
hcl
resource "aws_iam_user_policy_attachment" "user_attach" {
  user       = aws_iam_user.jack.name
  policy_arn = aws_iam_policy.developer_policy.arn
}
Attach to a group
hcl
resource "aws_iam_group_policy_attachment" "group_attach" {
  group      = aws_iam_group.devs.name
  policy_arn = aws_iam_policy.developer_policy.arn
}

Attach to a role
hcl
resource "aws_iam_role_policy_attachment" "role_attach" {
  role       = aws_iam_role.ec2_role.name
  policy_arn = aws_iam_policy.developer_policy.arn
}
