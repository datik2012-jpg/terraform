Why import is important
You use terraform import when:

you created resources manually in AWS

you inherited an AWS environment

you want Terraform to manage existing infrastructure

you want to avoid recreating production resources

you want to bring everything under IaC control

Terraform import is how you teach Terraform about existing infrastructure so Terraform can start managing it.
It’s one of the most important commands to master,  especially when you already have resources in AWS and want Terraform to take control.

Terraform does not create anything during import.
It only maps:

Terraform resource → AWS resource ID

After import, Terraform can manage the resource.



Write the Terraform resource block
Example: importing an EC2 instance:

hcl
resource "aws_instance" "ProjectB" {
  ami           = "ami-06178cf087598769c"
  instance_type = "t2.micro"
}
Terraform needs this block before import.

2️⃣ Run the import command
Example:

bash
terraform import aws_instance.ProjectB i-0abc123def456
This tells Terraform:

aws_instance.ProjectB → your Terraform resource

i-0abc123def456 → the AWS EC2 instance ID

Terraform writes this mapping into the state file.


3️⃣ Run terraform plan
This is where the magic happens.

Terraform compares:

the real AWS resource

your .tf configuration

If they differ, Terraform will show changes.

You must update your .tf code until plan shows no changes.

The private key is created in the same configuration directory we have been working on.



We have another EC2 instance created called jade-mw using the AWS CLI.

Using the AWS CLI inspect this EC2 instance and find the ID that is created by it.

Here are some of the specifications of this EC2 Instance:

AMI: ami-082b3eca746b12a89

Instance Type: t2.large

Key Name: jade

Remember the syntax to use the AWS CLI: aws command sub-command options

Also make sure to pass in the --endpoint http://aws:4566 (As we are using the AWS test framework. This is not required to interact with actual EC2 instances)

resource "aws_instance" "jade-wm" {
  ami           = "ami-082b3eca746b12a89"
